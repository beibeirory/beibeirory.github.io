sublime text 插件之Emmet

發表於 2016-03-18   |   分類於 前端开发   |  

以前曾在另一篇博文中记录了前端开发神器Webstorm，最近看到Sublime Text中的Emmet插件，觉得比webstorm还棒~~~ 这里主要简单记录下常用的语法结构。

Emmet的安装超级简单啦，跟其他sublime text插件一样一样滴。安装好以后，赶紧新建个html文件，感受下Emmet吧！

生成带有id、class的HTML标签

  • 带有id
    输入ul#aaa + Ctrl+E，则得到：
  • 带有class
    输入ul.aaa + Ctrl+E，则得到：

输入ul#ccc.ddd + Ctrl+E，则得到：

注：如果不给定标签的话，默认标签为div。如输入#aaa + Ctrl+E， 则会得到：

生成自定义属性 [attr]

输入a[href="http://beibeirory.github.io/"] + Ctrl+E，则得到：

其它标签和自定义属性都类似

对属性值编号 $

  • 一个$默认编号从1开始，多个$默认编号从0开始

输入ul>li.item$*3 + Ctrl+E，则得到：

输入ul>li.item$$$*3 + Ctrl+E

  • 实现倒序 @-
    输入ul>li.item$@-*3 + Ctrl+E
  • 从指定序号开始 @N
    输入ul>li.item$@3*3 + Ctrl+E

生成标签内容 {}

输入a[href="http://beibeirory.github.io\"]{Beibei's Blog} + Ctrl+E

注意：所有的输入中，都不能有空格
