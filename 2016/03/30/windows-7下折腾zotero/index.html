Q1：关于Zotero官方存储空间有限问题

主流处理方式当然是使用云端存储，实现Zotero中PDF文档和其他items分开存储。

具体做法：

- 选择合适的云端存储，这里强烈推荐[Dropbox](https://www.dropbox.com/)
不在赘述Dropbox的安装和注册，只说一点：如果你是一路默认安装的，可能会需要修改文件存储路径：这个功能在[首选项]-->[账户]-->[位置]。（如果你告诉我，你找不到首选项，那么可能Dropbox默认安装完就藏到你桌面右下角了，打开Dropbox面板找找那个“小齿轮”吧）
注：自然也有人推荐坚果云等，本人木有用过，不知道如何呢。

- 创建软链接 
将Zotero中的storage(D:\Zotero\storage)剪切到Dropbox目录(D:\Dropbox\Zotero)下，这样原来Zotero中storage就不见了，这时创建软链接到原来Zotero中storage的位置(D:\Zotero\storage)
（~~~~这里假设Dropbox路径是D:\Dropbox，Zotero路径是D:\Zotero\storage~~~）
**软链接的创建请参考Q2！**

- 设置Zotero
去你的Zotero中进行下同步设置：[工具]-->[首选项]-->[同步]-->[文件同步]下的复选框选择取消即可。

- Dropbox folder sync 
**注：**只是一种针对Dropbox的投机方法。（当时实在是各种方法在我的电脑上都不灵光，多亏伟大的博士师兄推荐了Dropbox folder sync，膜拜ing） 

说是投机，是因为Dropbox folder sync主要用来扩展Dropbox，使其能随意同步Dropbox目录以外多个任意指定文件夹的。但是由于其实现原理是将你指定的文件夹移动到 Dropbox 的主文件夹里面去，然后再做一个“符号链接”（类似快捷方式）放到原来的位置。嗯~~~我投机的用了这个**创建“符号链接”**的梗。。。
